## Challenge

Even very small games fail if the state is unclear. The user always needs to understand whose turn it is, what just happened, and what to do next.

## Solution

I kept the game interface minimal and focused on responsive board interactions, clear turn flow, and readable outcome states. Flutter made it easy to keep the UI clean while managing game state consistently.

## Outcome

Zing Tic Tac Toe is a compact but useful portfolio example because it demonstrates interaction design, state management, and front-end polish without needing a large codebase.
