## Challenge

Content apps become noisy quickly when live data, layout constraints, and API state all compete for attention. The main challenge was making external content feel readable rather than chaotic.

## Solution

I connected the app to REST APIs, shaped the data for mobile presentation, and kept the reading flow simple so headline-heavy screens stayed understandable. The work focused less on feature sprawl and more on reliability, structure, and responsive delivery.

## Outcome

This project improved my practical confidence with API integration, mobile content presentation, and the decisions needed to turn raw external data into a usable reading experience.
